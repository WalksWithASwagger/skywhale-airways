# Skywhale Airways — Duty-Free · Round 1

Merch designs (stickers + shirt prints) for *A Psychedelic Airport for Time
Travelers*. Generated with rafiki (Gemini 3 Pro Image), `style: none` so the
prompt carries the look, with the film's keyframes passed as reference for

delicate ink outlines, pastel palette (pale sky blue, dusty rose, golden yellow,
sage), warm and whimsical, dreamlike, slightly off. A *lost airline's* decals and
