# Skywhale Airways — Duty-Free · Round 5

Round 5 centers KK's new **I AM NOMAD.** line with the blonde **Time
Traveller** from the lost-baggage keyframe. Use one core art system and adapt it
across sticker, patch, and t-shirt so the set feels collectible rather than
scattered.

Source keyframe: `production/video_project/time_airport/keyframes/s02_baggage.png`

Generated by:

```bash
node scripts/create-r5-nomad-assets.mjs
npm run optimize
node scripts/diecut.mjs
```
